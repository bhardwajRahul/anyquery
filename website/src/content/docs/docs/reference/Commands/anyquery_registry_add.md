Add a new registry

```bash
anyquery registry add [name] [url] [flags]
```

### Examples

```bash
anyquery registry add internal_reg https://registry.anyquery.dev/v0/registry/
```

### Options

```bash
  -h, --help   help for add
```

### Options inherited from parent commands

```bash
  -c, --config string   Path to the configuration database
```
